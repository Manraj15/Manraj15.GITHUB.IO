Projects

CloudE

Github

CloudE is a decentralized web-application designed to allow users to bet ether (the Ethereum cryptocurrency) on various E-Sports games. All transactions happen securely on the Ethereum Blockchain. A smart contract written in Solidity automatically pays any winnings to users once a match ends.

TimeOut

Github

TimeOut is a Chrome extension designed to help users be more productive. The extension blocks and unblocks all pages ina given website according to time intervals set by the user. Users are also able to view their most frequented websites and the total amount of time they have spent on each site since the installation of the app in order to track their internet habits.

AmazonReads

Github

A Chrome extension designed to aid users in the process of buying a book from Amazon. The extension automatically displays reviews from GoodReads for any book the user is viewing on Amazon. The Extension also shows library availability of the book in the Toronto and Mississauga public library systems.

GOT-CastleBot

Github

This is a bot for the GameOfThrones subreddit. Using web scrapping, it gathers and displays data on any castle within the forum as a comment. Data displayed includes location, current house occupying the castle or keep, and a brief history of the castle and its owners.
